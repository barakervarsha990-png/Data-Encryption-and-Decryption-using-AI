# Data-Encryption-and-Decryption-using-AI
🔐 Data Encryption & Decryption Tool

A simple and interactive password-based encryption tool built using Streamlit.
This project demonstrates how basic encryption works using the XOR algorithm for both text and files.

🚀 Features
🔑 Password-based key generation using hashing
📝 Text Encryption & Decryption
📁 File Encryption & Decryption
💾 Download encrypted/decrypted files
⚡ Fast and lightweight UI with Streamlit
🔒 Session-based secure handling
🛠️ Technologies Used
Python 🐍
Streamlit
Hashlib (SHA-256)
XOR Encryption Algorithm

⚙️ How It Works
User enters a password
Password is converted into a secure key using SHA-256 hashing
XOR algorithm is applied to:
Encrypt text
Decrypt text
Encrypt files
Decrypt files
Output is displayed or available for download
💻 Installation & Setup
# Clone the repository
git clone https://github.com/your-username/encryption-tool.git

# Navigate to project folder
cd encryption-tool

# Install dependencies
pip install streamlit

# Run the app
streamlit run app.py
